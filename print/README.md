## Business Cards

**Specs:**
- Standard US: **3.5" x 2"** (89 x 51 mm) -- rectangular
- Square variant: **2.5" x 2.5"** (64 x 64 mm) -- rounded corners (rx=48)
- Resolution: 300 DPI (vector SVG scales perfectly)
- Two-sided: Front = warm paper with logo + tagline; Back = dark ink with contact info + scannable QR

**Print tips:**
1. Open `business-card-front.svg` and `business-card-back.svg` in your browser.
2. Right-click, "Save image as..." then save as PDF, or use **Print, Save as PDF**.
3. For professional print: open SVGs in Adobe Illustrator or Inkscape, add 1/8" bleed if your printer requires it, then export to press-ready PDF.
4. Print on thick cardstock (14pt or 16pt) with matte or soft-touch finish for best feel.

**QR code:** Both back designs include a real scannable QR code linking to `https://thevineathens.church` (version 2 matrix, SVG). Test by scanning with any phone camera app before printing.

## Hand Flyer

**Specs:**
- Half letter: **5.5" x 8.5"** (portrait)
- Contains woodcut logo, Ocoee landscape illustration, service info, and community group highlights.
- Dark editorial header band with warm paper body -- mirrors the site's light/dark section rhythm.

**Print tips:**
1. `flyer.svg` is camera-ready -- includes 1/8" bleed and crop marks.
2. Open in Illustrator/Inkscape, Export to PDF (300 DPI, include bleed).
3. For quick print: open in browser, Print, Save as PDF (trim manually if needed).
4. Print on 14pt cardstock or 80# cover paper for durability.

**Tip:** For bulk printing (church lobby, events), export the PDF and send to a local print shop (Staples, Vistaprint, etc.) with a request for "half letter cardstock, matte finish."

## Interest Form

**Specs:**
- Letter: **8.5" x 11"** (portrait)
- Visual form fields: Name, Email, Phone, Interest checkboxes (6), Notes textarea, Branch checkboxes (9+1 write-in).
- Print-ready: all fields render as pen-fillable boxes. No interactive submit (SVG is static for print).
- Warm paper background with white form fields for legible handwriting.

**Print tips:**
1. `interest-form.svg` is camera-ready -- includes 1/8" bleed and crop marks.
2. Open in Illustrator/Inkscape, Export to PDF (300 DPI, include bleed).
3. For pen/paper use: print on 24lb bond or heavier paper so handwriting doesn't bleed.
4. Form fields are white for legible handwritten responses. Place on lobby table with pens.

**Tip:** For digital fill, use the original HTML version or type onto a printed copy. The SVG is optimized for clean print output.
